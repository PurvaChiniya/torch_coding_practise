# torch_coding_practise

PyTorch and ML coding practice — from autograd fundamentals to full model implementations.

## Structure

```
core/           # Foundational concepts: autograd, backprop, attention, BPE, GPT training, batch norm
models/         # Full model implementations: decoder blocks, MOE, fine-tuning, vision transformer
rl/             # Reinforcement learning: blackjack, DeepRL (HuggingFace)
concurrency/    # Concurrency practice: distributed BFS, web crawler, fizzbuzz, basics
misc/           # Scratch notebooks, interview prep, data manipulation experiments
workflow_daily/ # Daily work tooling and session logging
data/           # Shared data files (names.txt, input.txt)
```
